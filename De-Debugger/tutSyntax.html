C++ Solutions.

Learn the basic syntax of the C++ language with our easy to follow cheatsheet.

Program Structure

#include statements at the beginning of the program, which allow access to the functionalities of the included libraries. main() function, which is run when the program is executed. return 0 at the end of the main() function, which indicates that the program should output 0 if it ran without issues


Comments

Comments do not affect the performance of a program because they are ignored by the compiler.

In C++, there are two types of comments:

  • Single-line: begin with //.
  • Multi-line: begin with /* and end with */.

Input and Output

Input and output make C++ programs more interactive.

Including the iostream library is necessary and must be placed at the beginning of the program to access input and output.

std::cout is the “character output” and it is used together with << to print to the terminal. std::cin is the “character input” and it is used together with >> to read user input. std::endl or \n can be used to insert a new line.


Variables

Used to store and retrieve data. When declaring a variable, it must be given a data type and a name. Variables of the same type can be declared in a single statement using a comma-separated list.

Variables can be declared with the const keyword, which prevents their value from being changed later.


References and Pointers

C++ provides two powerful features for memory manipulation:

  • & References: aliases to existing variables
  • * Pointers: store memory address as its value
Reference variables are created using the & symbol. & is also used to access the memory address of a variable.


Arithmetic Operators

C++ supports different types of arithmetic operators that can perform common mathematical operations:

  • + addition
  • - subtraction
  • * multiplication
  • / division
  • % modulo (yields the remainder)
